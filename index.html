Drago Clash

Game - Simple Online Game. Server EU. Game database using MongoDB.

Lobby - Access to all features, e.g. Shop, Costume Changer, Settings. Name also can be changed here.

Costume Changer - Face, Skin and Horn can be changed here. Normal wing also can be changed to special wing.

Shop - Offer lots of different and unique Face, Skin and Horn. Some items need In-App Purchasing.

In Game - There 2 types of mode in this game : Lives Mode and Star Mode. Check out the game to know more about that.

Video - A short video about this game. Get problems with publishing, but you can contact me if you want to try this game

Footprint Builder

Game - Constructing buildings using AR. Users can see CO2 consumption and money usage of all buildings.

Main Menu - Living standard selection. Higher living standard means higher cost of the materials.

Gameplay - Users point camera to referenced images and then a cube will spawn above that image.

Result - Shows CO2 consumption, money usage of all buildings and impact on earth.

Leaderboard - Show screenshot and rankings. Higher index means better performance

Video - A short video about this app.

Catastrophe

Game - 2-Player Cooperative Puzzle Game with QR and Speech Recognition.

Main Menu - Consists of feature that players can utilize.

Gameplay - Records of players' matches.

Result - Both players need to select with character the want to play as. Each character has different and unique skills.

Gameplay - Each character has specific dialogue and can only interact with specific objects. Check out the game to know more about that.
