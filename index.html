Footprint Builder


Group Project   •   Solo Programmer   •   Unity AR   •   Offline Mobile App   •   Architecture

Game - Constructing buildings using Unity Augmented Reality. Users can see CO2 consumption and money usage of all buildings.

Main Menu - Living standard selection. Higher living standard means higher cost of the materials. A fixed amount of money is provided for users.

Gameplay - Users point camera to referenced images and then a building will spawn above that image. Users can drag to adjust the height of the buildings.

Result - Shows CO2 consumption, money usage of all buildings and its impact on earth.

Leaderboard - Show screenshot and rankings. Higher index means better performance.

Video - A short video about this app prototype.

Catastrophe


Group Project   •   FirebaseSDK   •   MongoDB   •   SpringBoot   •   Speech Recognition

REST API   •   2-Player Co-op   •   Online Mobile Game   •   Puzzle Game

Game - online 2-player co-op puzzle game with QR and speech recognition.

Main Menu - Consists of features to create and join game. Moreover, you can take a picture in the game.

Match History - A record of completed matches. It displays the level and partner of that stage.

Result - Both players need to select with character the want to play as. Each character has different and unique skills.

Gameplay - Each character has specific dialogue and can only interact with specific objects. Check out the game to know more about that.
